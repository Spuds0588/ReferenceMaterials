SVG Filters Demonstration

Original Rectangle

<svg viewBox="0 0 200 200"> <rect width="100" height="100" fill="red" /> </svg>

Gaussian Blur

This filter applies a Gaussian blur to the rectangle.

<svg viewBox="0 0 200 200"> <defs> <filter id="blurFilter"> <feGaussianBlur in="SourceGraphic" stdDeviation="5" /> </filter> </defs> <rect width="100" height="100" fill="red" filter="url(#blurFilter)" /> </svg>

Color Matrix (Grayscale)

This filter converts the rectangle to grayscale using a color matrix.

<svg viewBox="0 0 200 200"> <defs> <filter id="colorMatrixFilter"> <feColorMatrix type="matrix" values="0.33 0.33 0.33 0 0 0.33 0.33 0.33 0 0 0.33 0.33 0.33 0 0 0 0 0 1 0" /> </filter> </defs> <rect width="100" height="100" fill="red" filter="url(#colorMatrixFilter)" /> </svg>

Drop Shadow

This filter adds a drop shadow to the rectangle.

<svg viewBox="0 0 200 200"> <defs> <filter id="dropShadowFilter"> <feDropShadow dx="5" dy="5" stdDeviation="3" /> </filter> </defs> <rect width="100" height="100" fill="red" filter="url(#dropShadowFilter)" /> </svg>

Combined Filters

This example combines a blur, grayscale conversion, and drop shadow.

<svg viewBox="0 0 200 200"> <defs> <filter id="combinedFilter"> <feGaussianBlur in="SourceGraphic" stdDeviation="3" /> <feColorMatrix type="matrix" values="0.33 0.33 0.33 0 0 0.33 0.33 0.33 0 0 0.33 0.33 0.33 0 0 0 0 0 1 0" /> <feDropShadow dx="5" dy="5" stdDeviation="2" /> </filter> </defs> <rect width="100" height="100" fill="red" filter="url(#combinedFilter)" /> </svg>
