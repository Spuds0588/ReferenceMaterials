What are information Tags?

To customize the email templates according to your need, you would need more information like First Name of the borrower, Last Name of the borrower, Email of the borrower, etc., so to help that we have introduced Information Tags, through which you can add more fields in your email templates.

How to use information Tags?

There are two ways to use information tags as follows:

  1. Click on Information tags button from the toolbar. A popup will open with the list of all tags. You can select any tag and press insert. Your selected tag will be inserted at cursor position in the editor

  2. Information tags are also provided as suggestions for ease of use. When you start typing an information tag manually which starts with {{. A list of tags will automatically be shown. You may keep typing for filtering. Select any tag and it will be autocompleted.
